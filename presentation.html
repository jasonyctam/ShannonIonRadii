Extending Shannon's Ionic Radii Database Using Machine Learning

Jason Tam

AI Reading Group 2021-07-08

Ionic Radius - I

  • Radius of monatomic ion in an ionic crystalline structure
  • $r_{cation}^{+}$ + $r_{anion}^{-}$ gives distance between nuclei
  • $pm$ ($10^{-12}$) or Å ($10^{-10}$)
  • Range from 31$pm$ to 200$pm$

Ionic Radius - II

  • Values depend on:
    • Oxidation State (OS)
    • Coordination Number (CN)
    • Orbital Configurations (OC)
  • Changes physical and chemical properties
  • Very useful for scintillators

Measurements

  • X-ray Crystallography (1912-)
    • Reconstruct structure from diffraction patterns
  • Landé (1920) found effective method with Lithium Iodide
  • Shannon performed major review (1976)

Machine Learning Modelling

  • Linear Regression
  • Support Vector Machines
  • Decision Trees
  • Ensemble of Decision Trees
  • Gaussian Process Regression (GPR)

Gaussian Process Regression

Results

  • GPR returned very good results
  • $R^{2} = 0.993$
  • $RMSE = 0.0332$Å
